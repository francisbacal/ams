# AMS - Asset Management System

This web application is for allocation and monitoring of company asset(s) to employees.
- User authentication and user registration (admin)
- Admin Authorization
- Asset Requests and monitoring

### Technologies used:
**» Laravel**
**» PHP**
**» MySQL**
**» jQuery**
**» Heroku**
**» remotemysql**
